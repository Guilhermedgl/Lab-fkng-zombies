
## ZOMBIES

My first project at Ironhack.

Kill zombies as much as you can or they will eat you!

++ play here: https://guilhermedgl.github.io/Lab-fkng-zombies/startercode/

## How to play

 + **W** - Up
 + **S** - Down
 + **A** - Left
 + **D** - Right
 + **Space-bar** - Shoot
